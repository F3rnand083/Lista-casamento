
# Site de Lista de Casamento (GitHub Pages)

## Como usar
1. Suba estes arquivos para um repositório no GitHub.
   - Se quiser URL raiz: nomeie o repositório como `SEU-USUARIO.github.io`.
   - Caso contrário, habilite **Settings → Pages** para a branch `main`.
2. Troque os links de pagamento (Mercado Pago / PagBank) no `index.html`.
3. Troque os números de WhatsApp (`NUMERO_A` e `NUMERO_B`) no script do `index.html`.
4. Substitua `img/foto-casal.jpg` por uma foto real do casal (JPEG ~2000px largura).
5. (Opcional) Atualize o iframe do RSVP com o seu Google Forms.

Pronto! O site estará online assim que o GitHub Pages processar a publicação.
